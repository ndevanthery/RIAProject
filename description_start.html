Par Simon Beaud le 05/04/2022 à 09:00

Lors du cours de RIA du 4ème semestre de la formation en informatique de gestion de la Hes-so Valais Wallis, nous devons réaliser Un projet en java script. Le but de ce projet est d'apprendre et d'appliquer les 3 langages du web (HTML, CSS & JS). Pour ce faire, il nous est demandé de développer un jeu sur base java script. Nous avons décidé de réaliser une version revisitée du célèbre SNAKE. Nous tenons à rendre à donner une dimension humoristique à ce jeu en y intégrant différents personnages
